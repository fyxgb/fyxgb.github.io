javaScript排序算法学习篇

发表于 2018-11-02 | 分类于 javascript ， 算法
字数统计: 928

冒泡排序：

普通冒泡排序

通过外层循环来控制比较次数，内层循环是进行数之间的比较，不推荐使用该方法，时间复杂度为O(n^2);

优化过的冒泡排序：

优化方法是利用后面的已经是排序好的数组，不需要再进行操作；

选择排序：

外层循环是进行比较次数，内层循环是通过进行数的比较；

插入排序：

外层循环进行比较次数，while是插入排序的关键，是将temp这个数组和前面的数组进行比较

归并排序：

巧妙利用递归原理和while进行循环比较，很实用的一个数组方法，时间复杂度为O（nlog^n）;

快速排序：

原理也就是取中间值，有条件到进行拆分，之后把左边最大的和右边最小的交换，执行到每个拆分的数组排序完成，得到一个排序好的数组，性能较好的一个排序方式，时间复杂度O(nlog^n)；
